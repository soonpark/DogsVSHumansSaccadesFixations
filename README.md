# Dogs vs. Humans: Saccade & Fixation Analysis (Work in Progress)
https://soonyoungpark.shinyapps.io/DogsVHumansSaccades/

Interactive R Shiny Dashboard for Comparative Eye-Tracking Data

1. Project Overview
   
This project provides an interactive visualization tool to explore differences in gaze patterns between dogs (Canis familiaris) and humans. This tool allows researchers to compare saccadic movements and fixation distributions across species.
